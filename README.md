# mac-symbol-cheatsheet

A small macOS menu bar utility that provides a searchable cheatsheet for macOS special characters and keyboard shortcuts.

## Motivation

New macOS users often need to look up special characters like:

* €
* ≤
* →
* •

and their keyboard shortcuts.

This tool keeps a searchable list directly in the macOS menu bar.

## Planned Features

* Menu bar utility
* Searchable list of macOS special characters
* Copy symbol to clipboard
* Categories (arrows, currency, math, typography)
* Favorites

## Tech Stack

* Swift
* SwiftUI
* macOS MenuBarExtra API
